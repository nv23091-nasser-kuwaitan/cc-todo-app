# CC Todo App — Polished UI

This is a small Flask todo application with a creative Picasso-inspired UI that has been refined into a clean, academic presentation suitable for a Harvard-style project demo.

Quick start

1. Create and activate a virtual environment (recommended):

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install requirements:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python app.py
```

4. Open http://127.0.0.1:5000 in your browser.

Files changed

- `templates/index.html`: major UI polish, fonts, layout, abstract SVG background, improved modal styling.
- `requirements.txt`: minimal deps.
